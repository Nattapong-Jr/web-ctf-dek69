# เฉลย Mini-CTF

เอกสารนี้รวบรวมขั้นตอนการแก้ปริศนาแอปพลิเคชันเว็บ mini-CTF อย่างละเอียด

## Phase 1: ด่านทางเข้า (SQL Injection)

1. ไปที่หน้าแรกของเว็บ (`/`)
2. คุณจะพบกับหน้าจอเข้าสู่ระบบแบบเทอร์มินัลที่ให้กรอก `USER_ID` และ `AUTH_KEY`
3. การกรอกรหัสผ่านทั่วไปจะขึ้นข้อความแจ้งเตือน "Access Denied" (การเข้าถึงถูกปฏิเสธ)
4. วิธีผ่านด่านนี้คือการกรอกโค้ด SQL Injection แบบพื้นฐานลงในช่อง Username หรือ Password
   - ตัวอย่างที่ 1: `' OR 1=1 --`
   - ตัวอย่างที่ 2: `" OR "1"="1`
5. คลิก **[ EXECUTE LOGIN ]** ระบบจะส่งคุณไปยังหน้าแดชบอร์ด

## Phase 2 & 3: แดชบอร์ดที่พัง และ ปริศนาเครือข่าย

1. เมื่อมาถึงหน้า `/dashboard` จะมีหน้าโหลดค้างอยู่ที่ "Fetching Payload Data... 99%"
2. UI จะจำลองข้อผิดพลาด (error logs) ขึ้นมาบนหน้าจอ และจะไม่แสดงข้อมูลใดๆ เพิ่มเติม
3. ให้เปิด **Developer Tools** ในเบราว์เซอร์ (ปกติจะกด `F12` หรือ `Ctrl+Shift+I` / `Cmd+Option+I`)
4. ไปที่แท็บ **Network** แล้วกดรีเฟรชหน้าเว็บอีกครั้งหากจำเป็น
5. มองหาคำขอ `fetch` ที่ร้องขอไปยัง `/api/sys-dump`
6. ตรวจสอบ **Response** (การตอบกลับ) ของคำขอนั้น คุณจะพบข้อมูล JSON ที่มีคำใบ้และรหัส Hex ต่างๆ (เช่น เลขที่อยู่หน่วยความจำที่สอดคล้องกับรหัส Unicode ของตัวอักษรภาษาไทย)

## Phase 4: การส่งคำตอบสุดท้าย

1. เมื่อคุณไขรหัสหาคำตอบสุดท้ายจาก JSON หรือคำใบ้ได้แล้ว (คำตอบที่ถูกต้องคือ **Nattapong นัทธพงศ์**)
2. กลับไปที่หน้า UI `/dashboard`
3. ที่บริเวณด้านล่างสุดของหน้าจอ จะมีช่องรับข้อมูลแบบเทอร์มินัลเล็กๆ ซ่อนอยู่ (`root@sys:~$`)
4. พิมพ์ `Nattapong นัทธพงศ์` ลงในช่องนั้น (ตัวอักษรภาษาอังกฤษพิมพ์เล็กพิมพ์ใหญ่ได้)
5. กดปุ่ม **Enter**
6. หน้าจอจะแสดงข้อความยินดี "SYSTEM OVERRIDE SUCCESS" พร้อมกับ ASCII art และข้อมูลการติดต่อแอดมิน
